<h3>题意<h3>
<p>给一个array，里面的元素都是出现两次的，除了其中有一个元素只出现了一次<p>

<h3>解题<h3>
<p>使用bit manipulation，异或。如果一个number和它自身异或，就变为了0<p>


```python
class Solution(object):
    def singleNumber(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        res = 0
        for i in range(len(nums)):
            res ^= nums[i]

        return res
```
